Reference

The reference documentation tells you about the components included on the latest version of Barebones, and how to use them. For checking how to download and install Barebones, read the getting started guide.

Typography

Barebones uses Roboto as the default font with Raleway used as the font for headers. The default font-size is 16px, and the default font-weight is 300 (light). Barebones comes with bold tag, italic, underline, links and small text. The default text color is #373d3f.

Buttons

Add a .button class to transform your div, a or button element into a normal button, or add the .btn--outline class to transform your element to an outline button.

Outline button
No Outline button

<div class="button--no-outline">No Outline button</div>

Grid

Barebones comes with flexboxgrid to provide a responsive and modern grid system. Please see flexboxgrid's documentation for how to use the grid.

Utilities

Barebones provides you with a number of utility classes:

Class Functionality
.u-float-right Float an element to the right
.u-float-left Float an element to the left/td>
.u-clearfix Clear floats
.u-show-on-print Show element when printing
.u-hide-on-print Hide element when printing
.u-page-break Force a page break when Printing
.u-full-width Make an element take up the full width
.u-full-width-max Give element a maximum with of the full width
.u-hide-on-tablets Hide element on tablet sized viewport
.u-hide-on-desktop Hide element on desktop sized viewport
.u-label--inline Display label input inline
.u-uppercase Transform text uppercase
.u-no-select Make element unselectable
