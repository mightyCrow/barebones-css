Getting Started

Download from GitHub

You can download a zip file with the latest release of Barebones here.

Using a CDN

You can use any of the cdns below to add Barebones to your project. Simply add it to the head section of your HTML

gitraw: 

<link rel="stylesheet" href="https://cdn.rawgit.com/tiaanduplessis/barebones-css/master/dist/barebones.min.css">

Using npm

You can use npm to download the latest version of Barebones. For this, open a new console window, navigate to you project home and run the command:

$ npm install --save barebones-css

This will download Barebones and place it on node-modules/barebones-css.

Note that you must have npm installed on your computer to download Barebones using this method.

Using Bower

You can use Bower for download your copy of Barebones. Same as npm, navigate to your project root folder and run the next command:

bower install barebones-css

You must have bower installed on your computer for use this option.

Clone the repo

Also, you can download Barebones cloning the repository hosted on GitHub. Simply open a new console window and run the next command: 

git clone https://github.com/tiaanduplessis/barebones-css.git

Add to your project

Now, if you have your copy of Barebones downloaded, the next step is adding the next line to the head of every HTML document where you want to use Barebones:

<link rel="stylesheet" type="text/css" href="path/to/barebones.css">

Additionally, you can add the minimized version of Barebones:

<link rel="stylesheet" type="text/css" href="path/to/barebones.min.css">

That's all! Now you can check the reference guide to start using Barebones.
